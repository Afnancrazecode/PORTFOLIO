SMART SHOPPING CART

The Smart Shopping Cart is an innovative, tech-powered solution designed to streamline in-store shopping. It uses an RFID-based system to automatically detect items placed in the cart and calculates the total cost in real time. Equipped with a load cell (HX711) for weight verification, touch sensor for final checkout, LCD display for billing, buzzer/LED indicators, and optional SIM800L module for SMS billing, this smart cart brings automation and accuracy into the retail experience. Powered by an Arduino Nano or ESP32, it ensures security through weight-based item verification and provides a seamless billing experience without waiting in queues. Ideal for smart retail environments, the cart enhances user experience while reducing human error and checkout time.
