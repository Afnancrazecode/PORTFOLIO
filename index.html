Smart Shopping Cart

The Smart Shopping Cart is a next-generation retail solution that automates billing using RFID technology and weight verification. As items are added, RFID scans the product, and a load cell (HX711) ensures accurate verification based on weight. An I2C LCD displays real-time billing while LEDs, buzzers, and a touch sensor guide the user through a hassle-free checkout.

Optional features like SIM800L integration for SMS billing enhance functionality. Built using Arduino Nano/ESP32, this smart cart minimizes queue time, prevents fraud, and enhances the in-store shopping experience.
